# Autocatch Poketwo

### How to use
###### Open the Terminal and digit
```bash
git clone https://github.com/Kameil/Autocatch-Pk2.git autocatch
```
```bash
 cd autocatch
```
```bash
pip install -r requirements.txt
```
```bash
pip uninstall discord.py-self
```
```bash
pip install discord.py-self
```
```bash
nano config.py
```
```bash
python main.py
```
